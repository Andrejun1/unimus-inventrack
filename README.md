# 🚀 UNIMUS Inventrack (UIT)

> *Smart Laboratory Inventory & Borrowing System*

---

## 📌 About The Project

**UNIMUS Inventrack (UIT)** adalah sistem berbasis web yang dirancang untuk membantu pengelolaan inventaris dan peminjaman barang laboratorium secara **digital, real-time, dan terintegrasi**.

Proyek ini dibuat untuk menggantikan sistem manual yang masih menggunakan buku, sehingga lebih **efisien, aman, dan transparan**.

---

## ❗ Problem

Beberapa permasalahan yang sering terjadi:

* 📖 Pencatatan masih manual (buku)
* ❌ Data sering hilang atau tidak rapi
* ⏱️ Tidak bisa monitoring stok secara real-time
* 🔍 Sulit melacak status barang
* ⌛ Tidak ada kontrol keterlambatan pengembalian

---

## 💡 Solution

UIT hadir sebagai solusi dengan fitur:

* 🌐 Sistem berbasis web (akses kapan saja)
* 🧾 Digitalisasi pencatatan peminjaman
* 📦 Manajemen data inventaris
* 🔄 Monitoring status barang (dipinjam / tersedia)
* ⏰ Reminder pengembalian
* 🔐 Penyimpanan data terpusat & aman

---

## 🎯 Objectives

* Meningkatkan efisiensi pengelolaan inventaris
* Mengurangi human error
* Meningkatkan transparansi data
* Mempermudah monitoring peminjaman
* Mendukung sistem digital kampus

---

## 👥 User Roles

| Role            | Description                            |
| --------------- | -------------------------------------- |
| 👨‍🎓 Mahasiswa | Melakukan peminjaman barang            |
| 👨‍💼 Admin     | Mengelola inventaris & data peminjaman |

## ⚙️ Planned Features

* **Deployment**: Vercel
